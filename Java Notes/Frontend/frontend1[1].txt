HTML 
=====
HTML stands for HyperText Markup Language.

It is widely used language on web to develop web pages and web applications.

It is developed by Burner's Lee in late 1991.

It is a tag based language.

HTML tag contains opening tag and closing tag.
ex:
	<html>  -- opening tag 
	</html> -- closing tag 

It is a case insensitive language.
ex:
	<HTML>
	<html>

HTML tags contain attributes and each attribute having name and value.
ex:
	<body bgcolor="red"> 

HTML tag may contains multiple attributes and each attribute is seperated with spaces.
ex:
	<body bgcolor="" background="">

A simple HTML document is known as component.

HTML is used to develop client side components.

All HTML documents we need to save with .html or .htm extension.

All HTML documents execute in a browser window.
ex:
	Chrome 
	firefox
	IE
	Microsoft Edge
	Opera
	Chromium
	and etc.

HTML documents we need to write in following editors.
ex:
	Notepad 
	notepad++
	Sublime
	Atom 
	VSC editor
	Dream viewer  
	and etc.

HTML 2.0 specification was released in the year of 1995.

HTML 4.0 specification was released in the year of 1999.

HTML5 which is a extension of HTML4.0 was released in the year of 2005.

The main objective of HTML5 is used to develop leight weight components.

HTML5 is also known as advanced hypertext markup language.

Note:
-----
	HTML does not support custom tags.	

HTML skeleton 
==============
<!DOCTYPE html>
<html>

	<head>
		- // head related tags 
	</head>
	<body>
		- // body related tags 
	</body>

</html>

Here <!DOCTYPE html> is used to represent HTML5 document.

<html>
------
It is a root tag for entire HTML document.

It contains two child tags.

ex:
	1) <head> tag 

	2) <body> tag 


<head> tag
----------
A <head> tag is used to declare following things.
ex:
	1) Page Title
	2) Favrouite Icon 
	3) Meta Data
	4) Styles 
	5) Scripts  

<body> tag 
----------
A <body> tag is used to declare actual content of our web page.
ex:
	images
	forms 
	tables 
	header 
	footer 
	and etc.
