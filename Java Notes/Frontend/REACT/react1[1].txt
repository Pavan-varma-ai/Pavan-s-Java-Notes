ReactJS 
========
It is a declarative, efficient and flexible javascript library that is used to develop frontend applications and User interfaces.

It is open source, component based javascript library responsible only for view layer of the application.

It is developed by Jordan Walke who was a scientist of Facebook.

It is initially developed and maintained by Facebook and later they have used in their own products like whatsapp and instagram.

The latest version of reactjs is v19.0.0.

The official website for react is https://www.react.dev.

It is used to create reusable components.

A component is a building block of react application.


Advantages of ReactJS 
=====================

1) Simple to learn and simple to use 

2) It is used to develop reusable components.

3) It supported by all major browsers.

4) It supports one way data binding.

5) Good documentation and community support.

6) It supports virtual DOM.
